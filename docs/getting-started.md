# Getting Started

Welcome to the Swiftx documentation! This guide will help you get up and running.

## Installation

To install Swiftx, run:

```bash
npm install swiftx
```

## Basic Usage

Here's a basic example:

```javascript
import Swiftx from 'swiftx';

const app = new Swiftx();
app.start();
```
